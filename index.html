Installation Guide

1

Download Files

Click the Install Game button to download the required files: freedom_city_v1.apk and freedom_city_resources.zip.

2

Install ZArchiver

If you don't have ZArchiver installed, download it from the Google Play Store.

Get ZArchiver
3

Extract Resources

Open ZArchiver and navigate to your Downloads folder. Find freedom_city_resources.zip and extract it.

4

Install APK

Go to your Downloads folder and tap freedom_city_v1.apk to install the game. You might need to allow installation from unknown sources in your security settings.

5

Play!

Open the installed Freedom City app and enjoy the game!
